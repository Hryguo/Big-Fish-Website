# Big Fish Chinese Cafe Website

A modern, professional website for Big Fish Chinese Cafe in San Jose, CA.

## ✨ Features

- ✅ Fully mobile responsive
- ✅ Fast and lightweight
- ✅ SEO optimized
- ✅ Google Maps integrated
- ✅ Click-to-call phone numbers
- ✅ Online ordering link
- ✅ Social media integration
- ✅ Professional design

## 📱 Sections

1. **Home** - Hero section with call-to-action
2. **Menu** - Visual menu with PDF links
3. **About** - Restaurant description
4. **Gallery** - Food photo showcase
5. **Contact** - Location, hours, phone, map

## 🎨 Customization

### Change Colors
Edit `css/style.css` at the top:
```css
:root {
    --primary-color: #d32f2f;
    --secondary-color: #c62828;
}
```
